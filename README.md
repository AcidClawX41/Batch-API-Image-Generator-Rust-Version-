# Batch API Image Generator (Rust Version)
API batch image generator with randomizer. Built with Rust + Slint. Generates AI images at configurable intervals with customizable prompt randomization (presets, orientation, lighting, camera, outfits, etc).

Supported APIs
xAI Grok Imagine (grok-imagine-image, grok-imagine-image-pro)
Google Gemini (planned)
OpenAI DALL-E (planned)
